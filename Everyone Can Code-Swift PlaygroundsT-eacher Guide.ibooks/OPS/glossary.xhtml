Coding

Coding is telling a computer what to do.

 

Developer

Developers write code to build their own apps and games.

Command

Command is a specific action for the computer to perform.

 

Sequence

Sequence is the order in which the commands are given.

Debugging

Debugging is the process of identifying and fixing the error.

Bug

A bug is an error in your code.

 

Call

Telling a program to run a function.

Composition

The process of combining small parts of a program to solve a larger problem.

 

For Loop

Runs a block of code over and over for a set number of times.

Function

A collection of commands grouped together and given a name. The set of commands can then be run with just the name of the function whenever that set is needed.

 

Logical operator

A symbol or words like “and,” “or,”and “not” that connects two or more Booleans to make conditional decisions more specific.

Boolean

A value that can only be either true or false.

 

Conditional code

A block of code that will run only if something is true. The "that" must stay in the definition.

Condition

Something you test that results in true or false.

While loop

A loop that runs a block of code as long as a given condition is true. When the condition is false, the loop stops running.

Pseudocode

Pseudocode is an informal description of code or a concept that’s intended for human reading.

Algorithm

An algorithm is a step-by-step set of rules or instructions.

 

Variable

A variable is a named container that stores a value. The value can change over time.

Type

Type is a named grouping of properties (the features) and methods (the behaviors) of a kind of data.

 

Initialization

Initialization is the act of creating a new instance of a type, which includes setting initial values for any properties of the type.

String

(text): A type that stores a series for characters, such as “hello world.”

 

Int

(numbers): A type that stores an integer—a number that has no decimal, such as 10 or -42

Bool

(true or false): A type that stores a value of either true or false.

Parameter

The ability to provide extra information for a function to do its work.

Escape sequence

In Swift, an escape sequence begins with the escape character. An escape sequence allows one or more characters to be used for purpose other than their typical use. For instance, the escape sequence of a backslash followed by a quotation mark changes the meaning of a quotation mark in a string from ‘mark the end the string’ to ‘insert a quotation mark character into the string’.

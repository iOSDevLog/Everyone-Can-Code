Build

Build means “assemble all the parts into an app.”

Run

To begin the execution of code. To launch or open an app is to run it.

Comment

A piece of text in a source code file that doesn’t get compiled as part of the program but provides context or other useful information about individual pieces of code.

Playground

A type of file in which you can change and play around with Swift code directly in Xcode and see the immediate results.

Results sidebar

Area within a playground where you can see the results of evaluating code in the source editor.

Assignment

An action that sets the value of a constant or variable.

Assignment operator

The = symbol used to set the value of a constant or variable. For example, greeting = “hello” sets the value of the variable greeting to “hello.”

Autocompletion

A feature of Xcode that infers what you’re trying to type from context and provides suggestions.

Camel case

The use of capital letters to indicate the beginning of a new words within multiword product and code names that do not use spaces or punctuation between words. Examples: QuickTime and viewControllerDidLoadRequiredFiles

Constant

A named container that stores a value that can’t be changed.

Characters

Unicode letters, numbers, and symbols.

Escape character

In Swift, the escape character in a string literal is the backslash `\` character. The escape character indicates that whatever follows it in the string will take on an additional meaning.

Quick Look button

In a playground, clicking the Quick Look button displays a result’s value in an appropriate representation. For example, viewing the value of an image constant or variable as an image.

Escape sequence

In Swift, an escape sequence begins with the escape character. An escape sequence allows one or more characters to be used for purpose other than their typical use. For instance, the escape sequence of a backslash followed by a quotation mark changes the meaning of a quotation mark in a string from ‘mark the end the string’ to ‘insert a quotation mark character into the string’.

String interpolation

The process of inserting string representations of constants, variables, literals, and expressions into longer strings.

Attributes inspector

An inspector used to customize visual attributes of a user interface element in a storyboard.

Image view

Displays a single image or a sequence of animated images in your interface.

Project navigator

Part of the Xcode workspace window that displays all the files in your project.

Simulator

An app in Xcode that simulates the behavior and appearance of running an app on a device.

Storyboard

A file that contains a visual representation of the app’s user interface, showing screens of content and the transitions between them, that you work on in Interface Builder.

Default answer

A predesignated value or setting used by a computer program when a value is not specified by the user.

Modulo

An operation that finds the remainder after one number is divided by another.

Comment in

To comment in is to add “//” before a line of code to tell the program to ignore the rest of the line and not to affect how the code is run.

Coding

The act of composing commands, code structures, and algorithms to create a computer program.

Comment out

To comment out is to remove the “//” in a line of code to implement how the rest of the line of code is specified. Also known as “uncomment.”

Concatenation

The operation of joining two strings together.

Conditional code

Code that runs only when specific conditions are met. For example, if statements and while loops contain conditional code that runs only if or while a condition is true.

Declaration

A piece of code that creates something new in your program, such as a function, constant, variable, or custom type.

 

Device

For purposes of this guide, “device” refers to iOS-enabled hardware: iPhone, iPad, iPad Pro, and iPod touch.

DNS

Abbreviation of Domain Name System, a protocol within the set of standards for how computers exchange data on the Internet.

Editable

A line of code that can be changed and is not fixed.

Encapsulation

A language mechanism for restricting access to some of the object components and/or a language construct that facilitates the bundling of data with the methods of operating on that data.

Interpolation

An estimation of a value within two known values in a sequence of values.

Main interface

The primary view of how a coder interacts with a computing platform to create a piece of software.

 

Manipulation

The ability to alter, edit, or change the constructs of text or data.

Navigator area

The area on the left side of the workspace window that lets you access your projects and workspaces.

Operator

A symbol, such as +, -, or &&, that represents an action on one or more values. For example, in 1 + 2, the addition operator (+) adds the numbers 1 and 2.

Pseudocode

A human-readable description of an algorithm, usually written using a mix of everyday language and code.

Scene

A storyboard representation of a screen of content in your app.

String

A type that stores a series of characters, such as “hello, world” or “dancing kittens.”

String parsing

The process of analyzing a string of symbols.

Swift

An open source programming language developed by Apple that’s used for writing iOS apps for iPhone, iPad, Mac, Apple Watch, and Apple TV.

Target

The instructions for building a finished product from a set of files in your project, for example, a framework, library, application, or command-line tool. Each target builds a single product.

Unresolved identifier

An error displayed by Swift when a name is not specified properly.

Value

A value is an expression which cannot be evaluated any further.

View

Objects that support drawing and event handling in Cocoa.

Xcode

An Integrated Development Environment (IDE)—tightly integrated with Cocoa and Cocoa Touch—where people can build apps for Mac, iPhone, iPad, Apple Watch, and Apple TV.

Xcode project

Contains the source files, libraries, media, and other resources needed to build a software product, such as an app.

Source editor

In a playground, an area where you type in Swift code and view inline Quick Look views.

Function

A named set of commands that can be run whenever needed.

Asset Catalog

A tool to manage assets like images that are used by your app as part of its user interface.

Calculation

A mathematical determination on the size or number of something.

Error

The occurrence of an incorrect result, visually displayed in Swift by a red error symbol.

Unicode

Unicode is the standard for digital representation of the characters used in writing all of the world’s languages, providing a uniform means of storing, searching, and interchanging text in any language.

Case

How a word is capitalized, for instance: upper case, lower case.

Iterate

To perform repeatedly.

Variable

A value that can change after it has been initialized, indicated in Swift by the var keyword.
